## This is a simple API created with [Express](https://expressjs.com/) and [Faker.js](https://github.com/marak/Faker.js/). 
The app runs on localhost:3000 and has 3 routes:
* '/' = Welcome page
* '/user' = Retrieves data of a single user
* '/users/:number' = Retrieves data of ':number' users (ex. /users/5 retrieves 5 users)
